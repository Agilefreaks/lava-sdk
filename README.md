# Lava-SDK!!
Welcome to Lava SDK 

# Installation: 

1. clone the directory. 
    ```
        git clone git@github.com:lavanet/lava-sdk.git
    ```
2. Install npm dependencies, compile to javascript, and convert to browser compatible JavaScript. 
    ```
        make build
    ```
3. Open dist/index.html from the browser.
